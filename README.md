# Marrow — Prototype

AI-powered product management platform for SaaS scale-ups.

## Run it

1. `cp marrow-config.example.js marrow-config.js`
2. Add your OpenAI API key to `marrow-config.js`
3. Open `index.html` in your browser (or run `python3 -m http.server 8080` and visit localhost:8080)

## Files

- `index.html` — full prototype (all pages, dark/light mode, accordion detail views)
- `marrow-prototype-v1.html` — previous version for reference
- `CONTEXT.md` — product context and design decisions
- `marrow-config.example.js` — API key template
